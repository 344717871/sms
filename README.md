# sms
sheepbao media server and super media server whith golang.

## Features
* modularization, expansibility
* high performance, easy deployment, cross-platform
* support rtmp 
* support http-flv
* support hls

## Build

```
git clone https://github.com/sheepbao/sms.git  
cd sms  
./build.sh  
```

## Run
./bin/sms
